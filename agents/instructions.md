# Agent Instructions — Knowledge Bank

Agents must treat this repository as the canonical research memory.

## Read order for company work

1. Relevant sector files.
2. Relevant frameworks.
3. Company README.
4. Company thesis, questions, evidence log and changelog.
5. Latest updates and sources.

## Write rules

- Reusable insight goes to sector/framework files.
- Company-specific insight goes to company files.
- Source-backed claims go to evidence log.
- Material opinion changes go to changelog.
- Raw notes go to notes, not thesis.

## Forbidden pattern

Do not recreate sector primers inside every company folder.
